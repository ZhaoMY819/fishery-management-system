# 钓手渔场管理系统

## 项目介绍
面向渔场经营者的一体化管理平台，采用前后端分离架构，涵盖用户管理、渔场管理、订单计费、数据统计等核心功能，实现渔场运营的数字化管理。

## 技术栈

### 后端
- **框架：** Spring Boot + MyBatis Plus
- **数据库：** MySQL
- **构建工具：** Maven
- **接口风格：** RESTful API

### 前端
- **框架：** Vue 3 + TypeScript
- **UI组件库：** Element Plus
- **状态管理：** Pinia
- **构建工具：** Vite
- **数据可视化：** ECharts

## 功能模块

### 后台管理端
- 用户管理：用户注册登录、角色权限控制
- 渔场管理：渔场信息增删改查、场地管理
- 钓手管理：钓手信息管理、排行统计
- 订单管理：订单查询、计费结算
- 数据统计：营收趋势、用户分析、可视化大屏
- 系统管理：菜单管理、角色管理

### 用户端
- 渔场浏览：查看渔场信息、场地情况
- 在线预约：预约钓位、计时计费
- 个人中心：订单记录、个人信息

## 环境要求
- JDK 1.8+
- MySQL 5.7+
- Maven 3.6+

## 运行方式

1. 创建数据库，导入 `db` 目录下的SQL脚本
2. 修改 `application.yml` 中的数据库连接配置
3. 执行 `mvn clean install` 安装依赖
4. 启动 Spring Boot 应用
5. 访问 `http://localhost:8080`

## 项目亮点
- 采用前后端分离架构，前后端独立部署
- 使用 Element Plus 构建现代化管理后台
- ECharts 数据可视化大屏，多维度展示运营数据
- 完善的权限控制体系
- 响应式设计，适配不同屏幕尺寸
